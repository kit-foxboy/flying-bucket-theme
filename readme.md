## About

Inspired by the "legendary" airship of the same name, now you too can command a flying auroran junkyard of the highest caliber! Picture yourself piloting an exotic alien craft through electric blue nebulae, with holographic displays casting ethereal glows across brushed metal surfaces with a mix of ancient and modern technology. Now powered by Matugen since I actually have no grasp of color theory whatsoever!

- **Chill Blue**: Relaxed blue for large bodies of text

## Installation
### VS Code
Launch VS Code Quick Open (Ctrl+P), paste the following command, and press enter:

```
ext install kit-kabbit.flying-bucket
```

### BetterDiscord
Launch BetterDiscord Settings, navigate to Themes, and click "Open Theme Folder". Place the downloaded theme file into this folder, then enable it in the BetterDiscord settings.

### Zed
Open Zed and hit `Ctrl+Shift+P` to open the command palette, then type `Install Dev Extension` and select the downloaded theme file. Alternatively, you can drag and drop the theme file into the Zed config folder. This is where you can find the config folder:

```
# On Linux
~/.config/zed/

# On Windows
%APPDATA%\Zed\config

# On macOS (I think anyway, might need to double check this at some point)
~/Library/Application Support/Zed/config
```
